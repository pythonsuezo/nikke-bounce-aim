================================================================================
  Nikke Bounce Aim（ニケ ボール反射エイム補助） 取扱説明書
================================================================================

【これは何？】
  NIKKE のミニゲームで、ボールが左右・天井の壁でどう反射するかを
  画面上に重ねて表示するオーバーレイツールです。
  ゲーム本体を改造するものではありません。

【動作環境】
  ・Windows 10 / 11（64bit）
  ・そのほかのソフトのインストールは不要（この exe だけで動きます）

【起動方法】
  1. NIKKE のミニゲームを開く（ウィンドウ表示推奨）
  2. NikkeBounceAim.exe をダブルクリック
  3. タスクバーに「Nikke Bounce Aim」が出れば起動成功です
     ※アプリを前面クリックで「アクティブ」にする必要はありません

【初回セットアップ（3クリック）】
  1. F1 を押す（または初回起動ですでに領域合わせモード）
  2. ① プレイ領域の左上（白い左枠の上あたり）をクリック
  3. ② プレイ領域の右下をクリック
  4. ③ 発射キャラの位置をクリック（高さの基準）
  終わると予測線が出ます。設定は同フォルダの config.json に保存されます。

【普段の使い方】
  ・マウスを動かす … その方向への反射予測線を表示
  ・ゲームはそのままクリック操作してOK（オーバーレイはクリックを透過します）

【発射点の合わせ方】
  キャラは横に平行移動するので、高さは一度決めればほぼ固定です。

  ↑ / ↓     … 発射点の高さ（Y）を微調整
  ← / →     … 発射点を左右に微調整
  Shift+矢印 … 5ピクセルずつ大きく動かす
  Alt + マウス左右 …
              高さを固定したまま、横位置だけキャラに追従
  F2         … 発射点をあらためてクリックで設定

【キー操作一覧】
  F1      プレイ領域の再キャリブレーション
  F2      発射点の再設定
  F3      軌跡の表示 ON / OFF
  F4      簡略表示（線だけ） / 詳細表示（ボール付き）切替
  , または 9    軌跡を薄くする（見やすく・透過）
  . または 0    軌跡を濃くする
  [ / ]         ボール半径を小さく / 大きく
  + / -         反射の描画回数を増やす / 減らす
  Esc           終了

【画面の見方】
  ・緑の線 … 予測軌道
  ・数字   … 反射の順番
      左の壁で跳ねた数字 → 左側に表示
      右の壁で跳ねた数字 → 右側に表示
  ・白いボール目印 … 今の「ボール半径」設定の大きさ
    ゲーム中の球と大きさが見た目で合うように [ ] で調整してください

【うまく合わないとき】
  ・ウィンドウ位置や解像度を変えた → F1 で領域をやり直す
  ・線の位置がズレる → 白い左右の枠にぴったり合わせて再キャリブ
  ・球の跳ね方が薄い/厚い感じ → [ ] で半径を合わせる
  ・画面がごちゃごちゃ → , で薄くする、または F4 で簡略表示
  ・紫一色でゲームが見えない → いったん Esc で終了し、最新版を入れ直す

【終了方法】
  Esc キー、またはタスクバーの「Nikke Bounce Aim」を右クリック → 閉じる

【セキュリティについて】
  初回起動時、Windows や Defender が警告することがあります。
  （個人作成の署名なし exe ではよくある表示です）
  「詳細情報」→「実行」などで起動できる場合があります。
  不安な場合は実行しないでください。

【注意】
  ・本ツールは非公式のプレイ補助です。利用は自己責任でお願いします。
  ・オンライン通信は行いません。設定は手元の config.json のみです。

【同梱ファイル】
  NikkeBounceAim.exe … 本体
  取説.txt           … この説明書
  config.json        … 起動後に自動作成（領域・発射点などの保存）
